### 这是一个centos脚本

最主要是给最小化安装的centos使用
在centos 7.5上测试通过

拥有下面的一些小功能

- 安装一些初始化的软件
- 安装python
- 设置静态ip
- 关闭firewalld
- 设置主机名
- 关闭selinux
- 安装docker
- 改变swap大小
- 安装oh-my-zsh
- 添加用户
